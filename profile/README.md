Berloga Studio is a small team building gaming communities from the ground up — custom infrastructure, plugins, automation, and web presence included.

## Projects

| Project | Description | Website |
|---------|-------------|---------|
| 🧱 **BerlogaCraft** | A network of modded Minecraft servers with custom modpacks, complex crafting systems, and full Russian localization | [berlogacraft.net](https://berlogacraft.net) |
| 🔫 **BerlogaRust** | A Rust game server with custom plugins and automated server management | [berlogarust.net](https://berlogarust.net) |
